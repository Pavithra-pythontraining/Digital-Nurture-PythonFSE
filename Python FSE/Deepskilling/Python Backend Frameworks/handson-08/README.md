# Handson-8: API Documentation using Swagger

## Objective

The objective of this hands-on is to generate interactive API documentation using Swagger UI and ReDoc.

---

## Technologies Used

- Django REST Framework
- drf-yasg
- Swagger UI

---

## Step 1: Install drf-yasg

```bash
pip install drf-yasg
```

---

## Step 2: Configure Swagger

Added Swagger configuration in `backend/urls.py`.

---

## API Documentation URLs

### Swagger UI

```
http://127.0.0.1:8000/swagger/
```

### ReDoc

```
http://127.0.0.1:8000/redoc/
```

---

## Features

- Interactive API Documentation
- Try API Requests
- View Request & Response Models
- JWT Authentication Support

---

## Learning Outcome

- Configured Swagger.
- Generated API documentation.
- Tested APIs directly from Swagger UI.
